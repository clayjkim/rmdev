# RM360 개발 일지

> 한국 의료기관을 위한 두 축의 소프트웨어를 개발하고 있습니다.
> ① **온라인 평판 모니터링 시스템** — 커뮤니티·지도·검색의 글을 자동 수집·분석해 위험/부정 여론을 실시간 감지·대응
> ② **통화보안·기록 앱** — 법인폰 통화를 자동으로 안전하게 보관하고, 통화 내용을 정리·분석
>
> 이 일지는 **개발 진척을 투명하게 공유**하기 위한 공개 기록입니다. 개인정보·고객사 정보는 포함하지 않습니다. **가장 최근 작업이 맨 위**에 옵니다.

---

## 2026-06-08

**통화보안 앱 (v0.9.5 → v0.9.6)**
- 통화 종료 후 기록 알림 흐름 재설계: 무시하면 **5분 뒤 한 번 더** 알림
- **연락처에 저장된 번호는 이름 자동 입력** (직원이 따로 안 적어도 됨)
- 놓친 통화는 나중에 **목록에서 일괄 입력**(미입력 건수 표시)
- 자동 보관 **정지 시 경고**, 앱을 강제로 닫아도 **자동 재시작**(직원 보호)
- 분류를 버튼식으로(클레임 강조), 통화 시점 표시
- **새 앱 아이콘** 적용

**평판 모니터링 시스템**
- 분류 정확도 개선: **부정 신호가 하나라도 있으면 '부정' 우선** 처리 (긍정 다수에 묻혀 부정을 놓치던 문제 해결)
- 검색 결과 **중복 수집 방지** (URL 표준화)

## 2026-06-01 ~ 06-04
